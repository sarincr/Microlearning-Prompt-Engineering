# Microlearning- Prompt-Engineering
This repository is a structured microlearning program for **Prompt Engineering**, designed for developers, researchers, and AI practitioners who want rapid, hands-on skill development.

The program is organized into **24 Sprints**, each focused on a specific capability within prompt engineering.  
Every Sprint contains **3–4 Skill-Drills**, which are bite-sized, applied exercises designed for immediate implementation.

---

## Focus Areas

The program emphasizes:

- Practical techniques  
- Clear mental models  
- Repeatable frameworks  
- Measurable improvement  
- Real-world prompt optimization  

---

## Suggested Usage

### Option 1 — Intensive Mode
Complete **1 Sprint per day** (24-day intensive)

### Option 2 — Steady Progression
Complete **2–3 Sprints per week**

### Option 3 — Reference Mode
Use the repository as a **targeted skill improvement library**
